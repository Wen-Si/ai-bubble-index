# AI Bubble Index | AI泡沫指数

全球AI泡沫指数实时监测平台 - 通过全球主要媒体报道的人工智能企业估值等关键数据进行计算分析。

Global AI Bubble Index real-time monitoring platform - calculated and analyzed based on key data such as AI company valuations reported by major global media.

## 功能特点 | Features

- 📊 **实时泡沫指数** | **Real-time Bubble Index** - 综合多维度数据计算的AI行业泡沫风险指数
- 💹 **企业估值追踪** | **Company Valuation Tracking** - 监控全球AI企业估值变化趋势
- 💰 **融资热度分析** | **Funding Heat Analysis** - 实时追踪AI领域融资动态
- 📰 **媒体情绪监测** | **Media Sentiment Monitoring** - NLP分析媒体报道情绪倾向
- 🌍 **市场指标整合** | **Market Indicator Integration** - 融合全球主要市场科技指数
- 🌐 **中英文双语** | **Chinese/English Bilingual** - 支持中英文界面切换

## 数据来源 | Data Sources

- Bloomberg / 路透社 / 金融时报 - 企业估值数据 | Company Valuation Data
- TechCrunch / Crunchbase / PitchBook - 融资交易数据 | Funding Transaction Data
- WSJ / NYT / 财新 - 媒体情绪分析 | Media Sentiment Analysis
- 纳斯达克AI指数 / 标普500科技 / 恒生科技指数 - 市场指标 | Market Indicators

## 技术栈 | Tech Stack

- HTML5 + CSS3 + JavaScript
- Chart.js - 数据可视化 | Data Visualization
- 响应式设计 | Responsive Design
