# AI Bubble Index | AI泡沫指数

全球AI泡沫指数实时监测平台 - 通过全球主要媒体报道的人工智能企业估值等关键数据进行计算分析。

## 功能特点

- 📊 **实时泡沫指数** - 综合多维度数据计算的AI行业泡沫风险指数
- 💹 **企业估值追踪** - 监控全球AI企业估值变化趋势
- 💰 **融资热度分析** - 实时追踪AI领域融资动态
- 📰 **媒体情绪监测** - NLP分析媒体报道情绪倾向
- 🌍 **市场指标整合** - 融合全球主要市场科技指数

## 数据来源

- Bloomberg / 路透社 / 金融时报 - 企业估值数据
- TechCrunch / Crunchbase / PitchBook - 融资交易数据
- WSJ / NYT / 财新 - 媒体情绪分析
- 纳斯达克AI指数 / 标普500科技 / 恒生科技指数 - 市场指标

## 技术栈

- HTML5 + CSS3 + JavaScript
- Chart.js - 数据可视化
- 响应式设计

## 部署

本项目部署在 Netlify 平台。
